UI del Sampler — dos superficies

Izquierda: el inspector del lane (pestaña Synth, para one-shots/keymap) — fila de knobs globales + keymap con filas compactas (nombre + root + rango + ✕) + mini-teclado de colores. Derecha: el editor de clip de forma de onda (para loop/tema) — waveform con trim arrastrable + modo Loop/Tema + campos BPM/ajuste/velocidad.
